2. User Dashboard View
3. Check Storage Status
4. Check Application Status
5. Fast Link Access
6. Fast App Open
7. Screen Snipper Tool
+++++++++++++++++++++++++++++++
1. How to run:
- Navigate to bin -> debug folder
- Run WindowsFormsApp2.exe
- Enter desired username, then click "Proceed" or press Enter
*Note: Make sure the AppList.csv, LinkList.csv and FileList.csv are 
present in the same foler as the executable

+++++++++++++++++++++++++++++++
2. User Dashboard view:
- Include: Clock, Calendar, CPU & RAM performance on the right side
- Other helper functions on the left side. More details below

+++++++++++++++++++++++++++++++
3. Check Storage status:
- Show the list of Drives and their freespace/total capacity

+++++++++++++++++++++++++++++++
4. Check Application status:
- Show the number and names of current running foreground applications 
- Doesn't include background applications

+++++++++++++++++++++++++++++++
5. Fast Folder Access:
- Show a list of Folders chosen by user
- Clicking on the Folder's name will open that Folder in Explorer
- Edit FileList.csv to add/remove/change folder list

+++++++++++++++++++++++++++++++
6. Fast Link Access:
- Show a list of URLS chosen by user
- Clicking on a link will open that URL in default browser
- Edit LinkList.csv to add/remove/change URL list

+++++++++++++++++++++++++++++++
7. Fast App Open:
- Show a list of Applications chosen by user
- Clicking on an App will launch that applications
- Edit AppList.csv to add/remove/change application list
- How to change AppList: provide the executable of the Application so 
the Helper can launch the app. 

+++++++++++++++++++++++++++++++
8. Screen Snipper Tool:
- Launch Snipping Tool to take screencuts.
